Project làm về web bán thiết bị điện tử


Các công nghệ chính :
   Backend dùng Springboot
   Fronend dùng Angular 4
   FrontEnd liên kết với backend theo chuẩn RESTful API

Các phần mềm cần thiết và cách cài đặt có thể xem ở file hướng dẫn cài đặt phần mềm.txt


Cách chạy chương trình (chỉ khi có đủ các phần mềm cần thiết) :
download file project và giải nén, click vào startProject.bat để chạy project hoặc
startanular.bat, startspringboot để chạy riêng angular. springboot
